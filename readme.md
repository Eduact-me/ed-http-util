# lucid-fetch-helper

## Installation

`npm install --save ed-http-util`

or using yarn

`yarn add ed-http-util`

## Usage

```javascript
import Http from "ed-http-util"

Http.respond(
    responseObject: any,
    data?: object,
    message: string = '',
    statusCode: number = 200,
    meta?: object,
  )
...
```
